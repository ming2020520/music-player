# 悦听音乐播放器 - API 接口文档

## API 基础地址

**当前使用：** https://api-enhanced-utik.onrender.com/

> 部署在 Render 平台，免费稳定

---

## 接口列表

### 1. 搜索歌曲

```
GET /search?keywords=歌曲名&limit=30

示例: https://api-enhanced-utik.onrender.com/search?keywords=周杰伦&limit=30
```

### 2. 获取歌曲播放URL

```
GET /song/url/v1?id=歌曲ID&level=standard

示例: https://api-enhanced-utik.onrender.com/song/url/v1?id=310574&level=standard
```

### 3. 获取歌曲详情

```
GET /song/detail?ids=歌曲ID

示例: https://api-enhanced-utik.onrender.com/song/detail?ids=310574
```

### 4. 获取歌曲评论

```
GET /comment/music?id=歌曲ID&limit=20

示例: https://api-enhanced-utik.onrender.com/comment/music?id=310574&limit=20
```

---

## 备用播放方案

当 API 返回的播放链接为空时，使用网易云外链：

```
https://music.163.com/song/media/outer/url?id=歌曲ID.mp3
```
